# Amani Hassan Portfolio Website

A modern, responsive portfolio website showcasing skills in Computer Science, UI/UX Design, and Human-Computer Interaction.

## Features

- **Clean, Modern Design**: Minimalist aesthetic with personality-driven elements
- **Fully Responsive**: Optimized for all device sizes
- **Accessible**: Follows WCAG 2.2 AA guidelines
- **Smooth Animations**: Subtle, purposeful animations and transitions
- **Interactive Elements**: Typing effect, smooth scrolling, hover interactions

## Sections

1. **Home**: Hero section with animated tagline
2. **About**: Personal bio and background story
3. **Skills**: Comprehensive tools and technologies showcase
4. **Projects**: Featured projects with case studies
5. **Resume**: Downloadable resume and key highlights
6. **Contact**: Contact form and social media links

## Tech Stack

- React 18
- JavaScript
- Tailwind CSS
- HTML5
- CSS3

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

### Update Personal Information

- Edit component files in `src/components/` to update:
  - About section bio
  - Skills and tools
  - Project details
  - Contact information
  - Social media links

### Styling

- Tailwind configuration: `tailwind.config.js`
- Global styles: `src/index.css`
- Color scheme uses primary purple/lavender and neutral tones

### Adding Your Resume

1. Place your PDF resume in the `public` folder
2. Update the resume download link in `src/components/Resume.js`

## Accessibility

This portfolio follows WCAG 2.2 AA guidelines:
- Semantic HTML
- Proper ARIA labels
- Keyboard navigation support
- Focus indicators
- Sufficient color contrast
- Alt text for images (when added)
